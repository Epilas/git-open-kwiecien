log

Przegladanie historii

git log [od ktorego hasha]

git log master

git log master~2

git log [do ktorego] --reverse

Limitowanie / filtrowani

git log -5

git log n4093qw4t -- plik.txt plik2.txt

git log --since="2 hours ago"

git log --after="2 hours ago"

git log --grep="tresc opisu commita"

--author=[pattern]

--committer=[pattern]

Commit Formatting

--pretty[=format] --format=format Formats oneline, short, medium, full, fuller, reference, email, raw, format:" moj format"

git log --all -- pokaz wszystkie galezie

git log --all --graph -- pokaz wszystkie galezie jako graf

git log --all --graph --decorate -- pokaz wszystkie galezie jako graf i oznacz referencje
