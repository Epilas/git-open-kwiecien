merge

Dolacza do bieacej galezi zmiany z innych galezi

git merge WIP_feature

git merge f39w40

git merge WIP_step1 WIP_Step2 WIP_fixy

Strategie

Fast-forward - kiedy galaz dolaczana MA W SOBIE galaz zrodowa - robimy "reset do przodu"

Merge commit - no-ff - tworzymy nowy commit "wezel" laczacy zmiany z 2,3,4... rodzicow

Git log

git log --all --graph --decorate

git log --all --graph --merges

git log --all --graph --no-merges
