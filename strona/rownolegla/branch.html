Branch

Etykieta wskazujaca na commit (i jego historie)

git commit - przesuwa aktywna galaz na najnowszy commit (budujac historie)

git checkout - ustawia nas na galezi (widzimy pliki z tej wersji)

git reset - ustawia galaz na inny commit (widzimy innna historie - z punktu widzenia tego commitu)

git branch - lista branchy

    WIP_logo
    WIP_stash
  * master

tworzy nowy branch

git branch nazwabrancha - utworz z bieżącej pozycji HEAD

git branch nazwabrancha [refCommit] - utworz na pozycji refCommit

git branch -d niezmergeowany - nie usuniemy commita ktory nie jest czesca innej galezi

git branch -D niezmergeowany - WYMUS usuniuecie commita ktory nie jest czesca innej galezi

Git checkout - przestawienie sie na branch

git checkout nazwabranch - przelacz na branch

git checkout -b nowybranch - utworz nowybranch i przelacz sie nań

Git switch - zmiana brancha (to samo co wyzej)

git switch nazwabranch - przelacz na branch

git switch -c nowybranch - utworz nowybranch i przelacz sie nań

git branch -m oldname newname
