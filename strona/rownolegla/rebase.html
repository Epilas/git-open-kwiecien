rebase

Zmiana bazy - resetujemy do wybranego rodzica i RE-aplikujemy commity na nowej bazie (rodzicu)

git rebase -i --- interaktywna lista zadan

# p, pick  = use commit

# f, fixup  = like "squash", but discard this commit's log message

Merge rebase

git rebase master - resetuje moje commity, mergeuje ff master na moj branch i reaplikuje moje commity

w przypadku konfliktu - rozwiazujemy, git add . , git rebase --continue
